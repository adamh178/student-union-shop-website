# Student Union Shop Website

# Description
Student union shop website for CO1707 Web Technologies assignment.

The website is being developed step by step, following the assignment brief and weekly lab work.

# Features
- Homepage with welcome content and branding
- Embedded HTML5 video
- Embedded YouTube video using an iframe
- Responsive navigation (desktop and mobile hamburger menu)
- Sticky footer

# Usage
Navigation links allow users to move between pages

# Validation
HTML and CSS validation will completed using W3C validation tools. Any unresolved errors will be commented here.
