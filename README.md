# Student Union Shop Website

# Description
Student union shop website for CO1707 Web Technologies assignment.

The site demonstrates responsive design and DOM manipulation using HTML5, CSS3 and JavaScript.

The website is being developed step by step, following the assignment brief and weekly lab work.

I focused on the use of responsive design, semantic HTML, DOM manipulation and client side interactivity. 

# Features
- Four page website:
    - Home Home (index.html)
    - Products (products.html)
    - Item (item.html)
    - Cart (cart.html)
- Semantic HTML structure using header, nav, main, and footer
- Embedded HTML5 video
- Embedded YouTube video using an iframe
- Responsive navigation (desktop, tablet and mobile hamburger menu)
- Sticky footer
- Product cards displayed using JavaScript
- Product filtering based on stock availability
- Item detail page populated using sessionStorage
- Shopping cart implemented using localStorage
- Ability to add items to the cart and clear the cart
- Sticky footer displayed consistently across all pages
- Scroll-to-top button for improved usability

# Usage
Open 'index.html' in your browser to use the website.
Navigation links allow users to move between pages.
Visit 'Products' to see a dynamicslly generated product list.
Click a product to view further details on the Item page.
Add products to the shopping cart.
View or clear the cart using the cart page.

# Responsive Design
Application is fully responsive and adapts to different screen sizes:
- Desktop - full navigation
- Tablet (<=768px) - adjusted spacing and sizing
- Mobile (<=600px) - hamburger navigation and sizing
